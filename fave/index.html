각종 버튼 가이드 [각종 버튼] [팝업 가이드]

class="m_btn" [공통 클래스]
- a / button 태그
data-btntype="button" [버튼 디자인]
- button : button 타입
- text : text 타입
- icon : icon 타입
hover="ripple" [마우스 오버 효과]
- ripple : 마우스에 따른 파장 효과
- none : 효과 없음 text type 용
- arrow : 아이콘 이동, 텍스트 체인지 text type 용
- underline : 아이콘 bg 체인지, 텍스트 라인 text type 용
size="l" [button type : 높이 기준 사이즈]
- l : 높이 48px
- m : 높이 56px
- h : 높이 64px
line="squire" [button type : 모양]
- squire : 사각형 버튼
- round : 라운드 버튼
color="black" [button type : bg color]
- black : 검은색 배경 버튼
- white : 흰색 배경 버튼
- none : Disable 버튼
icon="left" [아이콘 위치]
- left : 왼쪽 아이콘
- right : 오른쪽 아이콘
※ 위치에 따라 span.icon 의 돔 위치가 다름
※ 아이콘의 bg, 사이즈 등 기타 스타일 적용 필요
bg="w" [text type : 돔 bg]
※ 돔 BG에 맞는 컬러 입력
※ 돔 BG에 따라 아이콘 라인 컬러가 다름
- w : 돔 bg white → 아이콘 라인 투명도 X
- b : 돔 bg black → 아이콘 라인 투명도 O
data-txt="버튼" [text type : 버튼 copy]
※ hover arrow 용
※ span.txt 에 추가
- 버튼 카피와 동일하게 내용 작성
- 호버 용 카피

팝업 가이드 [각종 버튼] [팝업 가이드]

팝업 필수 항목 1. class = "m_popup"
2. id 필수 작성
3. class="pop_cnt_main" 에
data-lenis-prevent 필수
→ lenis 스크롤 해제 영역
팝업 열기 버튼 [ data-popup="POPUPSAMPLE" ] 속성 필수
※ POPUPSAMPLE = 팝업 아이디
팝업 닫기 버튼 [ class="m_pop_close" ] 필수 클래스
[ class="btn_close_pop" ] 닫기 아이콘 필수 클래스
그 외 [ class="m_pop_close-only" ] m_popup 두 개 이상 열렸는데 맨 앞의 한개만 닫고 싶을 때 (샘플팝업 2 참고)
[ class="dim none-click" ] 팝업 외 바깥 요소 눌러도 안 닫히게 해야할 떄
